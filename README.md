# API Google Maps

Aplicativo em Flutter para utilização das API's do Google Maps.

Substitua "YOUR API KEY" com a sua chave de api, nos seguintes arquivos:
  - android/app/src/main/AndroidManifest.xml
  - ios/Runner/AppDelegate.swift
  - lib/.env.dart
